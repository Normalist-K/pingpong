# Pingpong App

This is a Python-based sentence-making game using OpenAI API.

## Installation

1. Clone the repository:
```
git clone https://github.com/Normalist-K/pingpong
```
2. Install the required packages:
```
pip install -r requirements.txt
```

## Usage

To start the game, run:
```
python pingpong/main.py
```
